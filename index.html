BigdataSummer2015

Lab material for Big Data Course offered in Summer 2015 at NEU IS Grad Program.

View on GitHub Download .zip Download .tar.gz

Any Lab material will be update on this site. Students are requested to view this repository and clone the repo before the start of the Lab.

Each week we will be adding a new folder for that Lab. All the data files as well as the source code will be available to you. To clone it on your own machine use the following command :

$ git clone https://github.com/brahmbhattspandan/BigDataSummer2015.git

Any issues regarding any code file or general issues can be mailed @ brahmbhattspandan@gmail.com
